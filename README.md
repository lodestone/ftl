# Ftl

Faster Than Light. Fog Terminal Language. For Terminal Launching.

## Installation

Add this line to your application's Gemfile:

    gem 'ftl'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ftl

## Usage

### Example commands

    ftl list               #
    ftl start servername   #
    ftl connect servername #
    ftl stop servername    #
    ftl status servername  #
